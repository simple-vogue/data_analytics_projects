# Исследование бизнес показателей сервиса Яндекс.Афиша.

## Цель исследования
Целью проекта является анализ посещаемости, доходов и расходов сервиса Яндекс.Афиша, 
в результате которого были выявлены невыгодные источники трафика, определены перспективные когорты, 
сделаны выводы о финансовом положении сервиса и составлены рекомендации о перераспределении бюджета для умножения прибыли.

## Используемые данные
Доступны данные за период с июня 2017 по конец мая 2018 года:
- информация о посещениях сайта
- выгрузка всех заказов
- статистика рекламных расходов

## Библиотеки
*pandas, numpy, matplotlib, seaborn*
